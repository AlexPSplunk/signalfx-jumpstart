# POC in a Box

Clone this repository:

`git clone https://github.com/signalfx/poc-in-a-box.git`

Edit `variable.tf` with your *Access Token* and *Realm*

Initialise Terraform:

`terraform init`

Review the execution plan:

`terraform plan`

Apply the changes:

`terraform apply`

Destory all detectors

`terraform destroy`
